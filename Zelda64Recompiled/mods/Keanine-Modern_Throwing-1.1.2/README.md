# Modern Throwing
Inspired by the separated drop and throw actions in Breath of the Wild.

Need to throw a bomb while standing still? How about dropping a bomb while running? With this mod you simply press A to throw and B to drop! This functionality should work with anything Link can pick up.

Thanks to [tomtee](https://thunderstore.io/c/zelda-64-recompiled/p/tomtee/), this mod now includes UI action prompts for dropping and throwing!
